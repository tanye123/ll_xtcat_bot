  基于 LLOneBot 的 Python QQ 机器人插件框架

  功能特性

  - 通过 OneBot11 协议（WebSocket + HTTP）对接 LLOneBot
  - CustomTkinter 现代化桌面管理界面
  - 简洁的插件开发系统，支持消息/通知/请求事件处理
  - 插件热重载支持
  - 内置插件 API 服务器，支持插件 UI 配置界面
  - 完善的日志系统

  使用方法

  1. 确保 LLOneBot 已启动（默认 WS 端口 3888，HTTP 端口 3889）
  2. 运行 LL_XTCAT_Bot.exe
  3. 在设置页面配置连接参数
  4. 在插件管理页面启用/禁用插件
